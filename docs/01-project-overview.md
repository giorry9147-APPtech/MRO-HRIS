# MRO-HRIS Project Overview

## Doel
Een HR-systeem voor het Ministerie van Regionale Ontwikkeling.

## Wat het systeem doet
- Personeel registreren
- Functies beheren
- Afdelingen beheren
- Dossiers opslaan
- Salarisstructuur bijhouden
- Assets beheren (auto's, laptops, etc.)
- Rapportages genereren

## Kernlogica
Organisatie -> Functie -> Positie -> Medewerker -> Dienstverband -> Dossier -> Salaris -> Assets

## Doelgebruikers
- HR Administratie
- HR Manager
- Afdelingshoofden
- Directie
- Finance
- Assetbeheer

## MVP Scope
- Login
- Organisatiestructuur
- Functies
- Posities
- Medewerkers
- Dienstverband
- Documenten
- Salaris
- Assets
- Basisrapportages
